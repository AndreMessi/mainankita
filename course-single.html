Linux Untuk Pemula

  • Silabus Linux Untuk Pemula
    • Peserta dapat mengenal sistem operasi dan jenis-jenis OS termasuk linux yang banyak digunakan untuk kepentingan server dan lain lain.
    Peserta dapat menginstall sistem operasi linux berbasis ubuntu
    Peserta dapat memahami file system pada linux. Dapat menggunakan command line yang biasa di gunakan untuk kebutuhan Dapat menulis shell script
    Peserta dapat melakukan konfigurasi pada jaringan
    Peserta dapat melakukan manajemen pada user yang ada di OS linux
    Peserta dapat menggunakan ssh untuk melakukan remote login ke mesin serta dapat menjalankan perintah pada mesin server
    Peserta dapat melakukan manajemen pada penyimpanan hardisk dan sebagainya
    Peserta dapat melakukan pekerjaan otomatis menggunakan cronjob untuk menjalankan perintah atau script pada tanggal dan waktu tertentu secara berkala
    Peserta dapat melakukan installasi pada web service apache dan nginx
    Peserta dapat melakukan manajemen pada database mariadb/mysql
    Peserta dapat menjalankan aplikasi office pada linux Dapat mengoperasikan aplikasi multimedia pada linux
